infix'

Overview

This library provides a vocabulary for supporting simple infix math.

Expressions are processed from left to right.

Loading

The following should suffice:

needs infix'

Examples

with infix'

1 + 2 putn
4 * 2 + 3 putn

Functions

Name Stack Usage
+ ( x"-y ) 1 + 2
- ( x"-y ) 2 - 1
